# Pornire Aplicatie

Ruleaza aceste comenzi pentru pornirea completa a aplicatiei:

1) In terminalul pentru frontend (din root):
```
cd ./frontend/
```

2) Tot in terminalul de frontend:
```
composer run dev
```

3) In alt terminal, intra in `java-backend/` si ruleaza:
```
mvn -q spring-boot:run
```

4) Inca un terminal din root:
```
./simulate_sensors.sh
```
